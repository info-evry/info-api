EDT

Parameters

Name Type Description Example
level string The desired level. M1, L1
sublevel string The desired group. SR, G5
week integer The number of the week. 37
month integer The desired month. 05
year integer The desired year. 2020

List levels and groups

Get all levels and groups.

GET /v1/edt

List groups

List groups for the specified level:

GET /v1/edt/:level

Code group

Get EDT code and name for the specified group:

GET /v1/edt/:level/:sublevel

Response

Status: 200 OK: https://api.info-evry.fr/v1/edt/l1/g6

{
	"code":68426749,
	"name":"Groupe 6"
}

EDT Image

Get EDT data image for the specified group with week and year:

GET /v1/edt/:level/:sublevel/:week/:year
