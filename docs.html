API v1

  • EDT (GET Method)

    Get EDT.

    /api/v1/edt/{level}/{sublevel}/{week}/{year}
    • level: Level (str)
      ex. M1
    • sublevel: Group (str)
      ex. SR
    • week: Week (int)
      ex. 37
    • year: Year (int)
      ex. 2020

    Example

    /api/v1/edt/L1/G6/37/2020
